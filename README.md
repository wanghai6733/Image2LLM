# Image2LLM
截取图片并识别出图片的文字，最后将内容发送给LLM，获得答案解析。 (在release中可以下载exe版直接用)

# 依赖
* pip install pyQt5
* pip install QCandyUi
* pip install baidu-aip
* pip install openai

# 使用说明
- 可直接将待识别图片拖拽到该应用的空白处
- 也可以进行截屏识图, 按下底下的截屏按钮或者快捷键ctrl+alt+shift+F8
